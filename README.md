Personal Financial Logbook - SCIE Computer Science Project by Shunyao Zhang, David Li, Teddy Yuan, and Bruce Jin

Download all the files, put them in one folder, and double-click "home.htm" (open with a web browser) to run the web application.
**IMPORTANT!!** This web application uses **WebSQL** for databases, which unfortunately recent updates of Google Chrome and other browsers (Edge, Safari) no longer support. If the buttons are not working, **please install Chrome 97 or other browsers  that support WebSQL.**
